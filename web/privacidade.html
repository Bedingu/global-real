🔒 Política de Privacidade

Global Real — Plataforma de Investimentos Imobiliários

1. Introdução

A Global Real ("nós", "nosso") respeita a privacidade dos seus usuários. Esta Política de Privacidade descreve como coletamos, usamos, armazenamos e protegemos suas informações pessoais ao utilizar nosso aplicativo e website.

2. Dados que coletamos

Podemos coletar os seguintes tipos de dados:

  • Dados de cadastro: nome, email, telefone, empresa
  • Dados de perfil: interesses de investimento, orçamento, mercado preferido
  • Dados de uso: páginas visitadas, empreendimentos visualizados, funcionalidades utilizadas, tempo de uso
  • Dados de comunicação: mensagens enviadas no chat do app
  • Dados de assinatura: informações de pagamento processadas pelo Stripe (não armazenamos dados de cartão)
  • Dados de newsletter: email e idioma preferido
  • Dados técnicos: endereço IP, tipo de dispositivo, sistema operacional, navegador

3. Como usamos seus dados

Utilizamos seus dados para:

  • Fornecer e melhorar nossos serviços
  • Personalizar sua experiência no app
  • Qualificar leads com inteligência artificial para conectar investidores a oportunidades compatíveis
  • Enviar comunicações sobre novos empreendimentos e atualizações do app
  • Processar pagamentos de assinaturas
  • Gerar análises e relatórios internos (dados agregados e anonimizados)
  • Cumprir obrigações legais

4. Inteligência Artificial

Utilizamos inteligência artificial (Anthropic Claude) para:

  • Qualificar automaticamente o perfil de leads
  • Recomendar produtos de investimento compatíveis
  • Gerar mensagens de follow-up personalizadas
  • Calcular scores de compatibilidade

Os dados processados pela IA são tratados de forma segura e não são compartilhados com terceiros além do provedor de IA (Anthropic) para fins de processamento.

5. Compartilhamento de dados

Não vendemos seus dados pessoais. Podemos compartilhar dados com:

  • Supabase: infraestrutura de banco de dados e autenticação
  • Anthropic: processamento de IA para qualificação de leads
  • Stripe: processamento de pagamentos
  • Vercel: hospedagem do website

Todos os parceiros estão sujeitos a acordos de proteção de dados.

6. Cookies

Utilizamos cookies e tecnologias similares para:

  • Cookies essenciais: autenticação, sessão do usuário, preferências de idioma
  • Cookies opcionais: análise de uso, personalização de experiência

Você pode gerenciar suas preferências de cookies a qualquer momento através do banner de consentimento.

7. Armazenamento e segurança

Seus dados são armazenados em servidores seguros do Supabase na região sa-east-1 (São Paulo, Brasil). Utilizamos:

  • Criptografia em trânsito (HTTPS/TLS)
  • Row Level Security (RLS) no banco de dados
  • Autenticação segura via Supabase Auth
  • Chaves de API protegidas por variáveis de ambiente

8. Seus direitos (LGPD / GDPR)

Você tem direito a:

  • Acessar seus dados pessoais
  • Corrigir dados incorretos
  • Solicitar a exclusão dos seus dados
  • Revogar consentimento a qualquer momento
  • Solicitar portabilidade dos dados
  • Ser informado sobre o uso dos seus dados

Para exercer qualquer direito, entre em contato pelo email abaixo.

9. Retenção de dados

Mantemos seus dados enquanto sua conta estiver ativa ou conforme necessário para fornecer nossos serviços. Dados de newsletter são mantidos até que você solicite a remoção.

10. Menores de idade

Nossos serviços não são destinados a menores de 18 anos. Não coletamos intencionalmente dados de menores.

11. Alterações nesta política

Podemos atualizar esta política periodicamente. Notificaremos sobre mudanças significativas através do app ou por email.
